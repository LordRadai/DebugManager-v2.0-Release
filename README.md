### Prerequisites
A legitimate copy of Dark Souls II Scholar of the First Sin. Application version currently supported is 1.03. Regulation version doesn’t matter.
A controller to access the free camera and enemy control features. Any controller will work.
UXM to unpack the game. Refer to the UXM page for instructions on how to do it.

### Installing
Drag and drop Boot, DebugManager, font and the .properties files in your Game folder (i.e. where DarkSoulsII.exe is).
The regulation file is optional; it gives access to more options when choosing a player status param to use when map warping (refer to Map Selection for more information). If you want to use it, create a backup of the original regulation file from the Game folder (enc_regulation.bnd.dcx), rename the one provided to enc_regulation.bnd.dcx, then drag and drop over to the Game folder. Remember to revert to the original file when going online, or you will get banned.
DebugManager is a DLL mod. As such, you have two options for getting it to load:
1.
Use a chain loader, such as LazyLoader (recommended if using more than one DLL mod). Follow the specific loader instructions if using one.
2.
Using one of the DLLs Windows will automatically load upon program startup (dinput8.dll, dxgi.dll etc.). Rename DebugManager.dll to your DLL of choice, then drag and drop into the Game folder (where DarkSoulsII.exe is).
Once this is done, you can boot up the game. If you did everything correctly, it should boot up to a dark green screen with the mod name and version number on the top right, and a window called SceneSelect.
